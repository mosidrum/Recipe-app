# ğŸ“š Recipe App

Recipe App is a culinary recipe and grocery list management application with the following key features:

- **Login Page**: Allows users to log into their existing accounts.
- **Registration Page**: Enables new users to create accounts.
- **Food List**: Provides functionalities for adding, deleting, and querying food items.
- **Recipes List**: Supports the creation, reading, editing, and deletion of recipes.
- **General Shopping List View**: Displays missing ingredients for recipes and calculates the total price.
- **Public Recipes List**: Allows users to explore recipes shared by others.

The Recipe App is adaptable based on the development team's size, potentially leading to the addition of extra features.
